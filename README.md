# GramaticaDelLenguaje_2

Este proyecto tiene como objetivo determinar las posibles palabras del lenguaje de una gramática a partir de la información y especificaciones dadas por el usuario.

**Instalación**
Requerimientos
* Java 22 o superior
* Entorno de desarrollo Java (IDE) compatible con JavaFX

**Pasos de instalación**

1. Clona o Descarga el Repositorio
2. Descargar ZIP: Ve a la página principal del repositorio en GitHub. Haz clic en el botón verde "Code" y elige "Download ZIP". Una vez descargado, descomprime el archivo.
3. Clonar con Git: Si tienes Git instalado, copia la URL del repositorio. Abre la terminal o línea de comandos y ejecuta:
  * git clone https://github.com/
  * Reemplaza https://github.com/
4. Configurar el Entorno de Desarrollo
  * Verifica la Versión de Java: Asegúrate de tener Java instalado en tu máquina. Puedes verificar la instalación ejecutando       java -version en tu terminal o línea de comandos.
  *. Configura tu IDE: Abre el proyecto en tu entorno de desarrollo Java (Eclipse, IntelliJ, NetBeans, etc.). Asegúrate de tener JavaFX configurado y los complementos necesarios instalados.
5. Ejecutar la Aplicación
  * Abre la Clase Principal: Localiza y abre el archivo Principal.java en tu IDE.
  * Ejecuta la Aplicación: Ejecuta la clase Principal.java Esto iniciará la aplicación y abrirá la interfaz para que puedas           utilizarla.

*Siguiendo estos pasos, podrás clonar el repositorio desde GitHub, configurar el entorno de desarrollo y ejecutar la aplicación para usarla sin problemas. Recuerdar que es fundamental tener #Java# instalado y un entorno de desarrollo configurado para trabajar con proyectos JavaFX como este.*

**Uso**
* Una vez descargado el proyecto ejecuta el .jar para ejecutar la interfaz.
* Ingresa los valores solicitados para la construcción de la gramática.
* Haz clic en el botón Comprobar Gramática.
* Una vez verificadas las entradas ingresa la cantidad de palabras que deseas y el nivel máximo.
* Para obtener ayuda sobre cómo usar la aplicación, haz clic en el botón Ayuda o consultar la documentación.
  
**Estructura del Proyecto**
* GestorDeGramaticas: Realiza los llamados de las gramáticas, permite almacenarlas y manipularlas.
* Gramática: Contiene todos los métodos necesarios para gestionar la gramática y realizar la construcción de los árboles de derivación.
* Árbol de Derivación: Estructura de datos personalizada basada en un árbol N-Ario, permite la construcción de las palabras con base en una gramática.
